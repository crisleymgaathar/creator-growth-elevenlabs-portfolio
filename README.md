# Creator Growth Portfolio for ElevenLabs

This repository includes a creator growth portfolio developed for the ElevenLabs Creator Community Growth role.

The portfolio presents a strategic approach to building and scaling creator communities around AI-powered voice, audio, image, video, and storytelling tools.

## What this portfolio covers

- Creator ICP segments
- Sample AI creator workflow
- Community growth flywheel
- Workshop concept and event format
- Ambassador program framework
- Reporting cadence and community metrics
- 30, 60, and 90-day execution plan
- Product feedback loops for creator adoption

## Strategic focus

This portfolio is built around one central belief:

**Creators do not adopt tools because they are powerful. They adopt tools when they can see themselves making something better with them.**
